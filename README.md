# Custom Elements

This is a collection of various [custom elements](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) I developed for personal projects or during my work at [H&K+S](https://hks-agentur.de).

### [`<ajax-fragment>`](https://github.com/dweidner/ajax-fragment-element)

A custom element to enable partial page updates via AJAX

[Repository](https://github.com/dweidner/ajax-fragment-element) | [Demo](https://dweidner.github.io/ajax-fragment-element/)

### [`<opt-in>`](https://github.com/dweidner/opt-in-element)

A custom element to embed third party content with user consent

[Repository](https://github.com/dweidner/opt-in-element) | [Demo](https://dweidner.github.io/opt-in-element/)
